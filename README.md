# بورتفوليو عبدالله بسيوني — موقع متعدد الصفحات

موقع شخصي بلغتين (عربي/إنجليزي) بالكامل، دارك نيفي، بهوية الـ A بتاعتك، وهيرو ثلاثي الأبعاد (الجسيمات بتكوّن حرف الـ A — ولما تحرّك الماوس عليه بيتبعتر، وتبعّد بيرجع يتجمّع).

## 🗂️ الملفات
```
الموقع/
├── index.html        ← الرئيسية (الهيرو + الأرقام + 3 كروت)
├── about.html        ← عني (القصة + الرؤية + الرحلة)
├── work.html         ← الشغل (دراسات الحالة + حائط البراندات)
├── services.html     ← الخدمات (4 طبقات)
├── contact.html      ← تواصل
├── css/styles.css    ← التصميم والألوان
├── js/
│   ├── content.js    ← ✏️ كل النصوص هنا (ده اللي بتعدّله)
│   └── main.js       ← المنطق (مش محتاج تلمسه)
└── assets/
    ├── brands/       ← لوجوهات البراندات
    ├── fonts/ vendor/ ← خطوطك + مكتبات الأنيميشن (محلية)
    ├── logo-light.png ← لوجو الـ A الأبيض (ده اللي الجسيمات بتتشكّل عليه)
    └── favicon.png
```

## ✏️ تعديل النصوص
كل النصوص في `js/content.js`. كل حقل ليه `_en` و `_ar`. غيّر اللي جوه `" "` وبس، ومتغيّرش الأسماء اللي قبل النقطتين.
> أي كلمة بين أقواس `{ }` بتطلع مضيئة بلون مميز (في العناوين).

## 🖼️ تضيف أو تغيّر صورة
1. حُط الصورة في مجلد `assets/` (مثلاً `assets/brands/newbrand.png`).
2. في `content.js` اكتب مسارها.
   - لوجو براند في دراسة حالة: ضيف `logo:"assets/brands/newbrand.png"` جوه الـ case.
   - لوجو في حائط البراندات: في `brands:` ضيف `{n:"اسم البراند", l:"assets/brands/newbrand.png"}` — ولو مفيش لوجو اكتب `{n:"اسم البراند"}` بس.

## 🔗 تضيف لينك (مثلاً لخدمة)
أي عنصر ممكن يبقى لينك. أسهل مثال:
- **لينك في دراسة حالة:** ضيف `link:"https://..."` جوه الـ case في `content.js` (لو حبيت أفعّله في الكارت قوللي أوصّله).
- **روابط التواصل:** كلها في `meta:` (إيميل/واتساب/لينكدإن) — غيّرها من هناك وتتحدّث في كل الموقع.
- **أزرار الـ CTA:** نصوصها في `hero:` و `nav_cta`.

## 🎨 تغيير الألوان
في أول `css/styles.css` تحت `:root`:
```css
--ink-900:#0a0f18;   /* الخلفية الأغمق */
--blue:#4b6b8a;      /* اللون المميز */
--blue-lit:#7ea4c9;  /* لون النور */
```

## 🖌️ تبديل اللوجو/الصورة
استبدل `assets/logo-light.png` (نفس الاسم) — وده هيغيّر شكل الهيرو الثلاثي الأبعاد كمان لأنه بيتشكّل على هيئته.

## 🚀 الرفع على GitHub Pages
1. اعمل Repository جديد وارفع كل الملفات دي جواه.
2. Settings → Pages → Branch: `main` / folder: `/ (root)` → Save.
3. ملف `.nojekyll` موجود عشان كل حاجة تشتغل صح. مفيش build step — ارفع زي ما هو ويشتغل.

## 🆕 صفحات العملاء (دراسة حالة لكل عميل) — `case.html`

دلوقتي كل عميل ليه **صفحة مستقلة** بتفتح أوتوماتيك لما تدوس على الكارت بتاعه في صفحة `work.html`.
مفيش صفحة HTML لكل عميل — في صفحة واحدة بس اسمها `case.html` بتشتغل لكل العملاء عن طريق رقم العميل في اللينك:

```
case.html?id=aone      ← بتفتح صفحة A_ONE
case.html?id=mantorose ← بتفتح صفحة Mantorose
```

الـ `id` ده هو نفسه اللي مكتوب جوه كل عميل في `content.js` (مثال: `id:"aone"`).

### إزاي تكمّل بيانات صفحة أي عميل
كل الحقول دي **اختيارية** — ضيف اللي عندك بس، واللي تسيبه فاضي مش هيظهر. حطها جوه الـ `case` بتاع العميل في `js/content.js` (شوف A_ONE كنموذج كامل تنسخ منه):

| الحقل | بيعمل إيه |
|------|-----------|
| `about_en` / `about_ar` | فقرة «عن العميل» — مين هو والبراند بيعمل إيه |
| `duration_en` / `duration_ar` | مدة الشغل (مثلاً «90 يوم») |
| `services_en` / `services_ar` | قائمة «اللي قدّمته» (تظهر كـ tags) |
| `results_en` / `results_ar` | قائمة النتايج (لو سيبتها فاضية بيظهر `metric` العادي) |
| `gallery` | **المنشورات/الصور** (شوف تحت) |
| `pdfs` | ملفات PDF للتحميل/العرض |
| `links` | روابط مباشرة (إنستجرام، بيهانس… إلخ) |

### تضيف منشورات/صور لعميل
1. اعمل فولدر باسم العميل جوه `assets/work/`، مثلاً: `assets/work/kiin/`.
2. حُط فيه صور المنشورات (`jpg` / `png` / `svg`). أي مقاس — المربع والطولي بيتظبطوا لوحدهم.
3. في `content.js` جوه العميل اكتب:
```js
gallery:[
  {src:"assets/work/kiin/1.jpg", cap_en:"Ramadan campaign", cap_ar:"حملة رمضان"},
  {src:"assets/work/kiin/2.jpg", cap_en:"Launch post",     cap_ar:"بوست الإطلاق"},
],
```
> الصور بتظهر كمعرض، ولما تدوس على أي صورة بتكبر في عارض (lightbox) بأسهم يمين/شمال.

### تضيف ملف PDF (خطة، بروبوزال، تقرير…)
1. حُط الملف في فولدر العميل: `assets/work/kiin/plan.pdf`.
2. اكتب:
```js
pdfs:[ {file:"assets/work/kiin/plan.pdf", label_en:"Content plan", label_ar:"خطة المحتوى"} ],
```
> الملف بيفتح في تبويب جديد لما يتداس عليه.

### تضيف رابط مباشر
```js
links:[ {url:"https://instagram.com/...", label_en:"View on Instagram", label_ar:"شوفه على إنستجرام"} ],
```

> **ملاحظة:** كل النصوص لها نسخة `_en` و`_ar` زي باقي الموقع، فالصفحة بتشتغل بالعربي والإنجليزي وبتقلب RTL لوحدها.
